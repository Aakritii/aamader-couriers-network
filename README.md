# Parcel Tracking System

This is a fully working parcel tracking system with:

- Admin login using **real SMS OTP** (Twilio)
- Admin can **add/update status** and **upload delivery images**
- Users can track parcels in **Amazon-style UI**
- Node.js + Express backend
- JSON file storage
